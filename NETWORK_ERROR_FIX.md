# حل مشكلة Network Error

## المشكلة
عند الضغط على "بدء التقييم الطارئ" في الواجهة الأمامية، يظهر خطأ "Network Error".

## السبب
كان الـ Backend يسمح فقط بالاتصالات من المنفذ 3000، لكن Vite (الـ Frontend) يعمل على المنفذ 5173 افتراضياً.

## الحل ✅
تم إصلاح المشكلة! الآن الـ Backend يسمح بالاتصالات من كلا المنفذين:
- `http://localhost:3000`
- `http://localhost:5173`

## كيفية تشغيل المشروع

### الطريقة السهلة (موصى بها):
1. افتح PowerShell في مجلد المشروع
2. شغل الأمر:
```powershell
.\start.ps1
```
سيتم فتح نافذتين:
- نافذة للـ Backend (المنفذ 8000)
- نافذة للـ Frontend (المنفذ 5173)

### الطريقة اليدوية:

**نافذة 1 - Backend:**
```powershell
.\start-backend.ps1
```

**نافذة 2 - Frontend:**
```powershell
.\start-frontend.ps1
```

## التحقق من أن كل شيء يعمل

1. **Backend**: افتح http://localhost:8000/docs - يجب أن ترى واجهة Swagger
2. **Frontend**: افتح http://localhost:5173 - يجب أن ترى واجهة المشروع
3. جرب التقاط صورة والضغط على "بدء التقييم الطارئ" - يجب أن يعمل الآن بدون أخطاء! ✅

## ملاحظات
- تأكد من تشغيل الـ Backend **قبل** الـ Frontend
- إذا واجهت مشكلة CORS، تأكد من أن الـ Backend يعمل على المنفذ 8000
- إذا كنت تستخدم منفذ مختلف للـ Frontend، عدّل ملف `backend/utils/config.py`
